## 赞助能带来什么？

赞助将使项目可持续发展，用于：

- 新功能开发
- 错误修复
- 稳定性改进
- 技术支持

开源项目最宝贵的资源是维护者的时间。你的支持让我们能够持续为社区提供高质量的飞控固件。

## 已完成的里程碑

### Fydelix 项目 :material-new-box:

经过三年孵化，Fydelix 已完成：

- [x] 实时操作系统架构
- [x] 多 IMU 支持（BMI270、ICM-42688-P、QMI8658C）
- [x] 先进滤波系统
- [x] 完善的安全保护系统
- [x] MSP OSD 支持
- [x] Blackbox 日志
- [x] 多 RC 协议支持（CRSF、SBUS）
- [x] 多飞控板支持

### atbetaflight 项目

- [x] Betaflight 4.3.1 移植
- [x] F435 飞控支持（LQFP64）
- [x] F435 飞控支持（QFN48）
- [x] iNavFlight 移植
- [x] 传感器驱动优化
