## 为你的风格而生

**三种速率曲线**，可以混合使用：

- Betaflight Rate — 熟悉的手感，无缝迁移
- Actual Rate — 直观设置，所见即所得
- FlightOne Rate — FalcoX 用户的自然选择

Yaw、Pitch、Roll 可以分别使用不同曲线。你的飞行风格，由你定义。
