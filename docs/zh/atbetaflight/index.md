# atbetaflight

atbetaflight 是 Betaflight 的 AT32 移植版本，由 FlightNG 团队维护。

[查看源代码](https://github.com/flightng/atbetaflight){ .md-button .md-button--primary }

---

## 简介

atbetaflight 在 Betaflight 的基础上添加了对 AT32 芯片的支持，适合熟悉 Betaflight 的用户无缝迁移。

---

## 特性

基于 Betaflight 4.3.1，添加了以下功能：

- AT32F435/F437 芯片支持
- BMI270 驱动优化
- LSM6DSO 驱动优化
- 添加 LSM6DSL/LSM6DS3TR-C/LSM6DS3 支持
- 史密斯预测器

---

## 与 Fidelity X 的区别

| 项目 | atbetaflight | Fidelity X |
|------|--------------|------------|
| 基础 | Betaflight 移植 | 全新开发 |
| 配置工具 | Betaflight Configurator | CLI |
| 学习曲线 | 低（熟悉BF的用户） | 中等 |
| 功能更新 | 跟随 Betaflight | 独立开发 |

---

## 适用人群

- 熟悉 Betaflight 的用户
- 希望使用图形化配置工具的用户
- 需要 Betaflight 特定功能的用户
